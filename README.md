dash-back
=========

Repository for back-end codes for Dash project.

# Get Started

1. Set up a [virtualenv](http://www.virtualenv.org/) environment named `venv`
2. Activate it

    ```
    $ source venv/bin/activate
    ```

3. Install dependencies

    ```
    (venv)$ pip install -r requirements.txt
    ```

4. Enjoy!
